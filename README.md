# Tic-Tac-Toe Game 🎮

## Description
This is a terminal-based Tic-Tac-Toe game built in Python.
It includes:
- Normal Mode
- Competitive Mode (best of 3)
- Scoreboard
- Emoji interface (✖️ ⭕)

## How to Run
Run the file using:
python tic-tactoe-project.py

## Features
- Number-based board (1–9)
- Input validation
- Win/draw detection
- Competitive gameplay
